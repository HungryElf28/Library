# Библиотека - Онлайн читалка

Веб-приложение для чтения книг с функциями библиотеки: каталог, поиск, избранное, подборки, отзывы и режим чтения.

## Возможности

### Для всех пользователей (Гость)
- Просмотр каталога книг
- Поиск по книгам, авторам и жанрам
- Фильтрация по жанрам
- Просмотр деталей книги и отзывов
- Чтение книг

### Для авторизованных пользователей (Клиент)
- Все возможности гостя
- Добавление книг в избранное
- Создание собственных подборок книг
- Отслеживание прогресса чтения
- Создание закладок с заметками
- Написание отзывов

### Для администраторов
- Все возможности клиента
- Управление книгами (создание, редактирование, удаление)
- Управление авторами (создание, редактирование, удаление)
- Управление жанрами (создание, редактирование, удаление)

## Технологии

- React 18.3.1 + TypeScript
- Tailwind CSS v4
- Mock API с localStorage
- Context API для управления состоянием
- Pnpm для управления пакетами

## Запуск проекта

### Разработка

```bash
# Установка зависимостей
pnpm install

# Запуск dev-сервера
pnpm run dev
```

Приложение будет доступно по адресу: http://localhost:5173

### Production сборка

```bash
# Сборка проекта
pnpm run build

# Предпросмотр production сборки
pnpm run preview
```

### Docker

```bash
# Сборка и запуск через Docker Compose
docker-compose up -d

# Остановка
docker-compose down

# Просмотр логов
docker-compose logs -f
```

Приложение будет доступно по адресу: http://localhost:3000

## Особенности

### Темы
Приложение поддерживает светлую и темную темы с теплой цветовой палитрой (amber/orange) для комфортного чтения.

### Режим чтения
- Настройка размера шрифта (14-24px)
- Выбор шрифта (Georgia, Arial, Courier New)
- Межстрочный интервал (1.4-2.0)
- Три темы чтения: светлая, темная, сепия

### Подборки
Пользователи могут создавать свои плейлисты из книг, добавляя понравившиеся произведения в тематические подборки.

### Хранение данных
Все данные пользователя сохраняются в localStorage браузера:
- Избранное
- Закладки
- Прогресс чтения
- Подборки
- Настройки
